Handling of incomming packets depending on the type.

Try to parse the packet of bytes into a coap message and allocating ram.
Bad code/type combinations are silently ignored
Responds with 4.04 if the request handler can not be found.
Checking for critical options, NON messages are silently ignored.
try to include message into new or existing server/client interaction. Case RST, ACK, NON/CON.
